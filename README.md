Counter App

A Simple and beginner-friendly Counter Application built using **HTML**,**CSS** and **JavaScript**. This project is part of my learning journey to practice DOM manipulation and event handling.

Features:
-Increase the counter
-Decrease the counter
-Reset the counter
-Clean and responsive UI
-Smooth button hover effect

Technologies Used:
-HTML5
-CSS3
-JavaScript (DOM Manipulation)

Project Structure:
/counter-app
│── index.html
│── style.css
└── script.js

Preview:
A simple UI that displays the count and provides buttons to control it.

How to Run:
1. Download or clone this repository:
git clone https://github.com/your-username/counter-app.git
2.Open index.html in your browser

What I Learned:

-updating text content using Javascript
-Basic CSS layout & styling
-Handling increase, decrease and reset logic
